# BUYOWN - Laptop Shop Mobile App

This is a **React Native based E-Commerce Mobile Application** for a **Laptop Shop**.
The application allows users to browse laptops, view product details, and explore the store interface.

This project is developed as a **group project**, so all team members can collaborate and contribute through Git.

## Project Setup

Follow these steps to run the project:

1. Clone the repository
2. Open the project folder
3. Run the following commands:

-> npm install

-> npm expo start / npm start

## Running the App

After running `npm start`, Expo will start.

* Scan the **QR code** using the **Expo Go app** on your mobile phone.

OR

* Press **a** to run the project in **Android Studio emulator**.

Each member should:

* Pull the latest changes
* Create updates
* Push their work to the repository
